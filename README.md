 # News App

 ## App structure
 ```
  news-app
  ├── README.md
  ├── dummyData.json
  ├── loadDummyData.js
  ├── node_modules
  ├── package-lock.json
  ├── package.json
  ├── public
  │   ├── favicon.ico
  │   ├── index.html
  │   └── manifest.json
  └── src
      ├── actions
      ├── components
      ├── firebase.js
      ├── index.css
      ├── index.js
      ├── middleware
      ├── reducers
      └── styles
  ```

 ## Pre-requisites
  - `node version 8.9.4`

 ## How to Start the app
 ```
  npm install
  npm start
 ```

 - if you have any issues installing firebase or see any node-pre-gyp errors, try following
 ```
  npm i --save firebase --force
 ```
